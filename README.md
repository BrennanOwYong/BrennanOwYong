### 🧠 [memory-management-experiments](#)

Running GraphRAG, Neo4j, and vector DBs head-to-head and writing up honest takes.

> **18 Dec 2024** — GraphRAG is just recursive PCA dressed up as thorough
> summarisation. Pass.

> **14 Jan 2025** — Neo4j is great for relational data (obviously) but better
> as a table of contents than actual content storage and retrieval.

> **Final verdict** — Just use Mem0. It self-edits. Or MD files scoped per
> agent — that's what Claude Code does anyway.
